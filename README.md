# java-hangman
A Java implementation of the classic Hangman game demonstrating object-oriented programming, string manipulation, and user interaction
## Features
- Randomly selects a mystery word
- Tracks remaining lives
- Reveals correctly guessed letters
- Keeps track of previously chosen letters
- Case-insensitive letter matching
## Concepts Used
- Object-Oriented Programming
- Classes and Objects
- Arrays
- Loops
- StringBuilder
- User Input (Scanner)
- Randomization
## Language
- Java
